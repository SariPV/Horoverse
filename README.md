# Horoverse

Horoverse is an IOS Application using Swift language and Firebase Database. It is an application where it foretell the horoscope of people. There are many types of Horoscope options such as Birth Date prophecy, zodiac sign, and chinese horoscope. 

## Tools Used
 - Swift 
 - Firestore Database
 - AlamofireImage
 - IPhone 11
